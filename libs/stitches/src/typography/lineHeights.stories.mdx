# Line Heights

Line heights adjust the spacing between text lines, the noodle stitches library provides a set of values to adjust the line heights of a given element.

## Available Heights

- none
- tight
- snug
- normal
- relaxed
- loose

## Usage

export const code1 = `import { styled } from '@noodle/stitches';\n
const Text = styled('h1', {
  lineHeight: '$snug',
});\n
const App = () => (
  <Text>snug text</Text>
)`;
