# Typography Sizes

There are many typography sizes that are pre-defined and should be used to render text bigger or smaller, the sizes range from `xs` to `9xl`. The full order from smallest to biggest is `xs`, `sm`, `base`, `lg`, `xl`, `2xl`, `3xl`, `4xl`, `5xl`, `6xl`, `7xl`, `8xl`, `9xl`

## Usage

export const code1 = `import { styled } from '@noodle/stitches';\n
const Text = styled('h1', {
  fontSize: '$4xl',
});\n
export const App = () => (
  <Text>Hello World</Text>
);`;
