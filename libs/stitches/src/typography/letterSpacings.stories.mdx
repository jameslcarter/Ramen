# Letter Spacings

Letter spacings allows you to adjust the space between each letter on a given element, the noodle stitches library pre-defines a set of letter spacing values that you can use to adjust the spacing.

## Available Spacings

- tighter
- tight
- normal
- wide
- wider
- widest

## Usage

export const code1 = `import { styled } from '@noodle/stitches';\n
const Text = styled('h1', {
  letterSpacing: '$widest',
});\n
const App = () => (
  <Text>Wide text</Text>
)`;
