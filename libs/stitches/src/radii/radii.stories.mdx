# Radii

Included in the noodle stitches library is a set of pre-defined variables that can be used to adjust the border radius of an element. Currently there is `none`, `sm`, `normal`, `md`, `lg`, `xl`, `2xl`, `3xl` and `full`.

## Usage

export const code1 = `import { styled }  from '@noodle/stitches';\n
const Box = styled('div', {
  width: '$20',
  height: '$20',
  borderRadius: '$full',
});`;
